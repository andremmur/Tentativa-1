Super Trunfo C - DESAFIO

> Projeto desenvolvido como atividade prática da disciplina de *Introdução à Programação*.

O objetivo deste software é simular a mecânica básica do jogo de cartas "Super Trunfo", permitindo o cadastro de cartas e a comparação técnica de seus atributos para determinar um vencedor.

## 🛠️ Funcionalidades Implementadas
- *Cadastro de Cartas:* Entrada de dados via terminal para Nome, População, Área, PIB, etc.
- *Exibição de Dados:* Formatação clara das informações cadastradas para conferência.
- *Lógica de Comparação:* Algoritmo que identifica a carta vencedora com base em um atributo selecionado.

## 📊 Estrutura de Atributos
Cada carta no sistema possui os seguintes dados:
| Atributo | Tipo de Dado | Descrição |
| :--- | :--- | :--- |
| Estado | Char/String | Letra identificadora do estado (A-H) |
| Código | String | Código da carta (ex: A01) |
| População | Int/Long | Número de habitantes |
| Área | Float | Área em km² |

## 🧠 Lógica de Desenvolvimento
Para este projeto, foquei em garantir que a entrada de dados fosse robusta, para evitar erros de execução durante a comparação dos atributos.
